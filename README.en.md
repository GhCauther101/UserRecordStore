# UserRecordStore
web applicaiton for user record management options:
- create
- delete
- update
- manage tags

## Stack:
- Node - JavaScript execution evironment
- Vite - build system
- Vue 3 - UI web framework
- Pinia - state management framework for Vue.js
- TypeScript - script programming language

## Installation
1. Clone git repository:  
    `git clone https://github.com/GhCauther101/UserRecordStore`

2. Move to project folder:  
    `cd UserRecordStore/client/UserRecordStore.Web`

3. Install project dependecies:  
    `npm install`

4. Build the source project:  
    `npm run build`

5. Run up:  
    `npm run preview`
